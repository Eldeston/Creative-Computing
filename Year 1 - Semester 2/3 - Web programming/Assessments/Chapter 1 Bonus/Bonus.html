HTML Snippets

HTML snippets for code reference resource.

Header Tag

HTML headings are titles or subtitles that you want to display on a webpage. Tag starts with <h1> up to <h6>.

Paragraph Tag

A paragraph always starts on a new line, and is usually a block of text. Tag starts with <p>.

List Tag

HTML lists allow web developers to group a set of related items in lists.

Unordered List Tag

An unordered list starts with the <ul> tag. Each list item starts with the <li> tag. The list items will be marked with bullets (small black circles) by default.

Ordered List Tag

An ordered list starts with the <ol> tag. Each list item starts with the <li> tag. The list items will be marked with numbers by default.

Comment Tag

HTML comments are not displayed in the browser, but they can help document your HTML source code. A comment tag begins with <!-- Insert comment here -->

CSS Snippets

CSS snippets for code reference resource.

Inline Style

CSS styling can be implemented directly in HTML in a specific tag.

Internal Style

CSS styling can be implemented directly in HTML inside a <style> tag.

External Style

CSS styling can be implemented externaly in a .css file to be used by an HTML file.

Custom Classes

Classes can be used by elements to create a unique style that can be reused.
